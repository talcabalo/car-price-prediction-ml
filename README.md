# Car Price Prediction (Machine Learning Project)

End-to-end ML project for predicting car prices using real-world vehicle data.

## Features:
- EDA & Data Cleaning
- Feature Engineering
- Clustering (K-Means)
- Regression & Classification Models
- Model Deployment (HuggingFace)

## Results:
- R² = 0.99 (Regression)
- Accuracy ≈ 88% (Classification)
